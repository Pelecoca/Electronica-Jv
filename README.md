# Electrónica JV

Sitio web para Electrónica JV, especialistas en reparación y venta de equipos electrónicos.

## Características

- Catálogo de productos
- Carrito de compras
- Integración con WhatsApp para pedidos
- Servicios técnicos
- Diseño responsive
- Animaciones suaves
- Interfaz moderna y profesional

## Estructura del Proyecto

```
electronica-jv/
├── css/
│   └── styles.css
├── js/
│   ├── cart.js
│   ├── main.js
│   └── products.js
├── images/
├── index.html
└── README.md
```

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/electronica-jv.git
```

2. Abre el archivo `index.html` en tu navegador

## Uso

- Navega por los productos disponibles
- Agrega productos al carrito
- Realiza pedidos a través de WhatsApp
- Solicita servicios técnicos
- Contacta con la tienda

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- WhatsApp API
